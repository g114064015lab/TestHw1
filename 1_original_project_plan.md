# HW1: Simple Linear Regression with CRISP-DM

本作業的目標是撰寫一個 **Python 程式**，依照 **CRISP-DM 流程**來解決 **簡單線性迴歸問題**。同時要有 **Prompt**、**過程說明**，而不是只有程式碼與結果。程式需提供介面讓使用者修改 `a`（斜率）、`noise`（雜訊）、以及資料點數量，並使用 **Streamlit** 或 **Flask** 進行網頁化部署。

---

## Prompt（需求敘述）

請撰寫一個 Python 專案，需滿足以下要求：  
- 遵循 **CRISP-DM 流程**：Business Understanding、Data Understanding、Data Preparation、Modeling、Evaluation、Deployment。  
- 允許使用者輸入或修改 `a`（在方程式 `y = ax + b` 中）、`noise`（雜訊大小）、以及資料點數量。  
- 使用 **Streamlit** 或 **Flask** 架構，提供互動式網頁介面進行部署。  
- 程式需輸出 **分析過程**（步驟、數據摘要、圖表等），而不是單純顯示結果。
